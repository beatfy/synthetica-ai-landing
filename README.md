# Synthetica AI - Landing Page

Una landing page moderna y profesional para una consultoría de Inteligencia Artificial empresarial.

## 🚀 Demo

Ver en vivo: [https://beatfy.github.io/synthetica-ai-landing](https://beatfy.github.io/synthetica-ai-landing)

## ✨ Características

- **Diseño moderno** con gradientes y efectos visuales profesionales
- **Totalmente responsive** - Adaptado a móvil, tablet y desktop
- **Animaciones suaves** al hacer scroll
- **Formulario de contacto** funcional
- **SEO optimizado** con meta tags apropiados
- **Imágenes SVG** - Sin dependencias externas

## 📁 Estructura

```
synthetica-ai-landing/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # JavaScript para interactividad
├── assets/
│   └── images/
│       ├── hero.svg      # Imagen hero - Red neuronal
│       ├── process.svg   # Proceso de trabajo
│       ├── results.svg   # Dashboard de resultados
│       └── team.svg      # Equipo y expertise
└── README.md
```

## 🎨 Secciones

1. **Hero** - Título principal con estadísticas
2. **Logos** - Empresas que confían
3. **Servicios** - 6 servicios de IA ofrecidos
4. **Proceso** - Metodología de trabajo en 4 pasos
5. **Resultados** - Métricas de éxito
6. **Testimonios** - Casos de éxito de clientes
7. **Equipo** - 4 miembros del equipo directivo
8. **Contacto** - Formulario de consulta gratuita
9. **Footer** - Links y contacto

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 con variables CSS y Flexbox/Grid
- JavaScript vanilla (ES6+)
- SVG para gráficos vectoriales
- Google Fonts (Inter)

## 🚀 Despliegue

### GitHub Pages

1. Ir a Settings > Pages
2. Seleccionar rama `main` y carpeta `/ (root)`
3. Guardar y esperar a que se genere

### Otros hosts

Simplemente subir los archivos a cualquier servidor web estático.
